Managing User Permissions

Project Creation and Management ››


A primary concern of researchers and data mangers is data control. VOEIS provides highly granular user access controls that allow PIs to grant varying degrees of read and/or write permissions to any registered VOEIS user. After you create a new project in VOEIS, you are able to select who may use your project and grant various levels of read/write priveleges each individual.
