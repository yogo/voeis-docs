Understanding the Concept of the Variable


Before endeavoring to manage variables in a Project, it is important that to understand the concept of the variable in VOEIS. A variable in VOEIS is more than just a variable name like 'Temperature'. A variable is an entire collection of information (or metadata) about an individual or multiple data values that includes the following:


  • Variable Name
  • Units
  • General Category
  • Speciation
  • Data Type
  • Sample Medium
  • Quality Control Level
  • Spatial Offset Type
  • Spatial Offset Value
  • Spatial Offset Units
  • Time Collection Interval
  • Collection Interval Units
  • Field Collection Method
  • No Data Value
  • Sensor Type
  • Logger Type


These fields are modeled after the conceptualization of the variable in the Observation Data Model (ODM) used by the CUAHSI-HIS. The terms associated with each of these fields and avaialble for use in the creation of variables in any VOEIS project is discussed in the next section. More information about each of these fields may be found in the ODM documentation here. It is strongly recommended that users review this documentation. It is critical that users understand the concept of the variable.
