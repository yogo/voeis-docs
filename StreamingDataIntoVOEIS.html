Streaming Data Into VOEIS


Many research groups maintain networks of observation sensors that stream data at a fine time scale to one or more aggregation points. Manually loading this data is often unfeasable due to the high frequency of data collection. VOEIS accomodates researchers who want to automate the data upload process for this sort of data by offering an extensive set of APIs that can push data to the Data Hub. While use of the APIs does require some programming experience, they are relatively simple to use and client-side file-managment scripts that use the APIs are generally straightforward to develop. Users interested in using the APIs are encouraged to read the VOEIS User's Manual.
