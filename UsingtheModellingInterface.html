Integrating Simulation Models

Advanced Topics ››


The VOEIS framework is designed to allow data resources to be accessed by a wide variety of simulation models. VOEIS can feed input or boundary condition data to the simulation and the new data and metadata produced by simulations can subsequently be added to the data resource library. Users who wish to use this feature must simply integrate one or more pre-built APIs into their simulation model.
